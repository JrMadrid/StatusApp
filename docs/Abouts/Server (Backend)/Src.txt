# SRC

Este directorio contiene el código fuente principal del proyecto.

Estructura general:
- Cada subdirectorio agrupa componentes, controladores o utilidades específicas del sistema.
- Se sigue una estructura modular para mantener el código organizado y fácil de mantener.

Convenciones:
- Nombres de archivos y carpetas en minúsculas.
- Cada módulo incluye su propio archivo README.txt si es necesario.
- Cada documento debe tener un titulo al inicio. 

Notas adicionales:
- Solo debe incluirse código fuente aquí, no archivos de configuración ni documentación externa.
